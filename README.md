MHFS Vimfiles
=============

My VIM settings and plugins.

Installation
------------

    git clone git://github.com/mhfs/vimfiles.git ~/code/vimfiles/
    git submodule init
    git submodule update

    ln -s ~/code/vimfiles/ ~/.vim
    ln -s ~/code/vimfiles/vimrc ~/.vimrc
    ln -s ~/code/vimfiles/gvimrc ~/.gvimrc
